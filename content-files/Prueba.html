La estructura de datos tipo Series es un ndarray unidimensional cuyos elementos o datos pueden ser de diferente clase y que están asociados a los índices del array. Por defecto, el primer índice del array es de valor 0 y, el último, N-1 (donde N es el número de elementos).

Una de las características principales de las series es que el alineamiento entre datos es intrínseco: la asociación entre los índices y los datos no puede romperse a menos que sea de forma intencionada. De esta forma, se asegura que las operaciones entre series se realicen de forma correcta, es decir, entre los mismos índices. Esta característica permite realizar operaciones basadas en la vectorización, que ofrecen un mayor rendimiento y se consideran una buena práctica de programación.
